# Thiranex-E-commerce-web-application
# E-Commerce Web Application

## Project Description
This is a simple E-Commerce Web Application developed using frontend, backend, and database technologies. The application allows users to view products, add items to cart, and place orders. It also includes basic admin functionalities for product and order management.

---

# Features

- User Login
- Product Display
- Add to Cart
- Order Management
- Admin Dashboard
- Backend API Integration
- Database Connectivity

---

# Technologies Used

## Frontend
- HTML
- CSS
- JavaScript
- React.js

## Backend
- Node.js
- Express.js

## Database
- MySQL / MongoDB
